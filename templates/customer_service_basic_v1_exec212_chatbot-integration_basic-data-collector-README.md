#### AKTIVIERUNG
- Workflow in N8N oeffnen und auf "Aktivieren" klicken
- Webhook-URL aus dem ersten Node kopieren: `/customer-service-chatbot-integration`

- POST-Request an die Webhook-URL senden

```json

  "focus": "chatbot-integration",
  "priority": "normal",
  "source": "Ihr System"

```
